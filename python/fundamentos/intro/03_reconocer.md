## Declaración de variable
- Permite asignar un valor a un nombre para su uso posterior en el programa.


## Revisión de tipo
- Verifica el tipo de dato asignado a una variable (por ejemplo, usando `type()`).


## Revisión de tamaño
- Verifica la longitud o cantidad de elementos en un objeto (por ejemplo, usando `len()`).


## Comentarios
- **Una línea**: se utiliza `#` al inicio de la línea. 
- **Múltiples líneas**: se utilizan triples comillas simples (`'''`) o dobles (`"""`) para encerrar el bloque de texto.

## Bonus: Errores Comunes
- **NameError**: `name <variable name> is not defined`
- **TypeError**: `'tuple' object does not support item assignment`
- **KeyError**: `'altura'`
- **IndexError**: `list index out of range`
- **IndentationError**: `unexpected indent`
- **AttributeError**: `'tuple' object has no attribute 'pop'`
- **AttributeError**: `'tuple' object has no attribute 'append'`


> **Nota**: Algunos de estos errores pueden surgir al manipular tipos de datos de manera incorrecta, por ejemplo, intentando modificar elementos en una tupla. En Python, las tuplas son inmutables, es decir, no se pueden cambiar ni agregar o borrar elementos después de su creación.
